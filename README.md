# Bluetooth-Enable-Mobile-Manipulator
Designed by Pravalika Kota, Jampu Brahma Teja, and Lokesh Jaya rao, this Mobile Manipulator integrates a 4WD chassis with a 4-DOF robotic arm. Utilizing a dual-Arduino (Uno &amp; Nano) architecture, it ensures stable Bluetooth control for remote navigation and precise pick-and-place tasks, effectively bridging mobility with manipulation.
